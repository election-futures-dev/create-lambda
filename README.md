# create-lambda
Create AWS Lambda using AWS CLI

## Prerequisite
- [ ] [Install AWS CLI v.2]
```bash
$ curl "https://awscli.amazonaws.com/AWSCLIV2.pkg" -o "AWSCLIV2.pkg"
$ sudo installer -pkg AWSCLIV2.pkg -target /
```

- [ ] [Configure your AWS credentials]
```bash
aws configure
```
